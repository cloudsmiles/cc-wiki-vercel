# 一行多列求和

| field | comment |
| --- | --- |
| name | 姓名 |
| sex | 性别 |
| chinese | 语文成绩 |
| math | 数学成绩 |
| english | 英语成绩 |

求男女成绩第一名，并且打印语数英三科成绩，使用sql语句

思路：使用select max(…)  … group by sex的方式

SQL语句：

```jsx
select max(math+chinese+english),name,sex,chinese,math,english from student group by sex;
```
