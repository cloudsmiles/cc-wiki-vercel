# srp概述
SRP 是 Unity 的可编程渲染管线系统，它允许开发者：
- 完全控制渲染流程
- 自定义渲染策略
- 优化特定平台的性能

# Unity 提供的三种渲染管线

- 内置渲染管线 (Built-in)
    
    - 传统的固定渲染管线
    
    - 功能固定，不可自定义
    
    - 适合简单项目
    

- 通用渲染管线 (URP)
    
    - 基于 SRP 的轻量级解决方案
    
    - 针对移动平台优化
    
    - 适合大多数项目
    

- 高清渲染管线 (HDRP)
    
    - 基于 SRP 的高端渲染方案
    
    - 提供高质量图形
    
    - 适合 PC/主机高端项目

# SRP 的优势

1. 可定制性

- 完全控制渲染流程

- 自定义渲染顺序

- 添加特殊效果


2. 性能优化

- 移除不需要的渲染步骤

- 针对特定平台优化

- 批处理控制


3. 功能扩展

- 自定义光照模型

- 自定义后处理效果

- 自定义阴影技术

# 注意事项

1. 性能考虑

- 注意渲染顺序

- 控制DrawCall数量

- 优化shader复杂度


2. 兼容性

- 考虑多平台支持

- 处理不同硬件能力

- 提供降级方案


3. 调试

- 使用Frame Debugger

- 监控性能指标

- 处理异常情况
