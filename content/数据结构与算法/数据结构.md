# 数据结构

| Name | 关键词汇 |
| --- | --- |
| 数组 | 多读少写，存储连续，访问随机 |
| 链表 | 少读多写，存储随机，访问连续 |
| 栈 | 类似于羽毛球桶，FILO。入栈，出栈，栈底，栈顶 |
| 队列 | 类似于单行隧道，FIFO。入队，出队，队头，队尾                                    循环队列 —- 底层数组实现，队尾可以在队头前面 |
| 散列表 | key, value, hash函数                                                     hash冲突：开放寻址法，拉链法                                        操作：读写，扩容 |
| 树 | 高度，子节点，父节点，根节点                         |
| 二叉树 | 查找 —— BST, left比父节点小，right比父节点大       遍历： 前序，中序，后序，层序                                  DFS（前中后）—— 递归|栈，BFS（层序）—— 队列 |
| 二叉堆 | 最大堆，最小堆                                                               操作：插入 —— 放在最后一个位置，上浮                   删除 —— 删除顶节点，最后一个位置交换，下沉        构建 —— 非叶子结点下沉                                             优先队列，二叉堆实现 |
|  |  |
