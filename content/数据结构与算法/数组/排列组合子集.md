# 排列组合子集

三种变化

- 元素不重复，不可复选，比如 nums = [1, 2, 3]，3只有一种组合
- 元素重复，不可复选，比如 nums = [1, 2, 2, 3]， 4有[2, 2], [1, 3]
- 元素不重复，可复选，比如 nums = [1, 2]，3有[1, 1, 1], [1, 2]

三种问法

组合，子集，排列

所以总共有九种题型

**但无论形式怎么变化，其本质就是穷举所有解，而这些解呈现树形结构，所以合理使用回溯算法框架，稍改代码框架即可把这些问题一网打尽**
