# 滑动窗口

算法框架

```jsx
/* 滑动窗口算法框架 */
void slidingWindow(string s, string t) {
    unordered_map<char, int> need, window;
    for (char c : t) need[c]++;

    int left = 0, right = 0;
    int valid = 0; 
    while (right < s.size()) {
        char c = s[right];
        // 右移（增大）窗口
        right++;
        // 进行窗口内数据的一系列更新

        while (window needs shrink) {
            char d = s[left];
            // 左移（缩小）窗口
            left++;
            // 进行窗口内数据的一系列更新
        }
    }
}
```

例题

- 最小覆盖子串 (if valid == len(need),  才缩小左空间，子串包含t的元素，长度最小）
- **字符串的排列  (if right - left ≥ t.Size(), 才缩小空间，子串长度固定）**
- **找到字符串中所有字母异位词**
- **无重复字符的最长子串 (if window[c] ≤ 1, 才缩小空间，子串等无重复字符，才固定）**

思路

need和window两个map，need记录需要匹配的元素及次数，window记录窗口内对应的元素及出现次数

valid，用来记录子串符合的情况（valid == len(need)）

left, right，窗口的边界情况，先right++，找到子串后，再left++，缩减空间
